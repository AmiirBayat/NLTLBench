End-to-End Abstraction-Based Control from Natural Language
with LLM-Enhanced NL-to-LTL Translation

Amir Bayat1, Raphaël M. Jungers1
1ICTEAM, UCLouvain, Louvain-la-Neuve, Belgium
Paper arXiv Code Benchmark Video

Abstract

Abstraction-Based Controller Design (ABCD) offers a principled framework for the safe control of complex Cyber-Physical Systems (CPSs), but interfacing real-world requirements with its formal synthesis machinery remains a major bottleneck. In practice, such requirements are most naturally expressed in Natural Language (NL), whereas ABCD requires formal specifications such as Linear Temporal Logic (LTL).

This paper presents an LLM-enhanced pipeline for ABCD, in which NL requirements are translated into LTL and used within a formal synthesis workflow. We implement the pipeline in Dionysos.jl and introduce a benchmark for evaluating lifted NL-to-LTL translation under both logical diversity and linguistic variation.

Experiments with state-of-the-art LLMs show that translation accuracy degrades systematically with the complexity of the target specification, measured by AST size and temporal depth. These results reveal a scaling relationship between LLM success rate and LTL formula complexity.

Overview

The proposed framework translates NL requirements into LTL formulas, validates the generated formal specification with the user, constructs the corresponding Büchi automaton, and synthesizes an ABCD controller. The abstract system is computed offline from the CPS model.

Benchmark Construction

We introduce a benchmark for evaluating NL-to-LTL translation beyond a limited set of hand-written templates. LTL formulas are generated with controlled AST size, back-translated into canonical English, and then paraphrased using multiple LLMs to introduce linguistic diversity.

Results

We evaluate several state-of-the-art LLMs on the generated benchmark under zero-shot and few-shot prompting. The results show that performance decreases as the structural complexity of the target LTL formula increases.

Success rate as a function of LTL formula size, Büchi automaton size, and temporal depth.

Interactive NL Interface for ABCD

We implement an interactive interface for synthesizing ABCD controllers from NL requirements. The user enters an NL requirement, the system generates the corresponding LTL formula, and the controller synthesis pipeline is executed through Dionysos.jl.
